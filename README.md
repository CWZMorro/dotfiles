# Here are my collection of dotfiles
powered by stow

## Usage

### 1. Install stow 
 For Arch: `sudo pacman -S stow`

 ### 2. Clone the repo in home directory
  `git clone https://github.com/CWZMorro/dotfiles.git`

  ### 3. Change directory to the dotfiles directory
  `cd dotfiles`

### 4. To use the app's config:
 `stow APP_NAME` (Exp: qtile, bash, picom.....etc)
